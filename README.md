### 🎉 二次开发

1. 拉取代码

```bash
$ git clone https://github.com/hanfengju888/better
$ cd better
```

2. 安装依赖

```bash
# 可换豆瓣源或者清华源安装依赖
$ pip install -r requirements.txt
```

3. 安装并启动redis

4. 配置mysql(修改根目录下config.py文件)

6. 启动服务

```bash
$ python run.py
```

7. 注册用户

打开浏览器输入: `http://localhost:7788`进入登录页。
